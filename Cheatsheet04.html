Box Model

Block elements have which kind of characteristics?
Their content is displayed in a new line
Inline elements have which kind of characteristics?
Their content is inlined - displayed between other content in the line
What is the margin of an Html element? Where can one find it
A margin is the space outside something
What is the padding of an Html element? How does it differ from margin?
Padding creates extra space within an element while margin creates extra space around the element

Classes and Ids

Write down a selector for a certain class
.headline
Write down a selector for a certain html element of a certain class
h2.headline
Write down a selector for a certain id
#certainid
Write down a selector for a certain html element of a certain id
p#certainid
Write down the start tag of a certain html element with a specific class.
<h2 class="headline">
Write down the start tag of a certain html element with a specific id.
<p id="certainid">
Do classes select one or more elements?
Classes usually select more elements
Do ids select one or more elements?
Ids usually select one element (only one id per element and just one occurence per id per html-page)
How often can classes / ids occur on one page?
An html element can only have one unique id, while classes can be used on multiple elements (not unique) and multiple classes can be assigned to one element
